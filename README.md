# Slay List

A cute, pink-themed task manager for macOS built with SwiftUI.

## Features

- Natural language task input ("buy groceries tomorrow 3pm", "meeting in 2 days !high")
- Projects, tags, and smart lists (Inbox, Today, Upcoming, Logbook)
- Kanban and list views
- Pomodoro timer
- Calendar integration
- Stats dashboard with streaks and completion rates
- Auto-updates via Sparkle

- **SlayList-1.1.zip** — standalone app

## Installation

1. Download the DMG or ZIP from the link above
2. If using the DMG, open it and drag **Slay List** into **Applications**
3. If using the ZIP, unzip and move **Slay List.app** to your Applications folder
4. **First launch only** — since the app isn't signed with an Apple Developer certificate, macOS will block it. To open:
   - Right-click (or Control-click) on **Slay List** in Applications
   - Click **Open** from the menu
   - Click **Open** in the dialog that appears
   - After this one-time step, the app opens normally forever

   Alternatively, run this in Terminal:
   ```
   xattr -cr /Applications/Slay\ List.app
   ```

## Auto-Updates

Once installed, Slay List checks for updates automatically. You can also check manually via the menu bar: **Slay List → Check for Updates...**

## Requirements

- macOS 14.0 (Sonoma) or later
